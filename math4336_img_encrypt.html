Image Encryption, Decryption of Radon Transformed Data

Github link by BEN AYED, Ahmed; LEE Wing Hang; LI Yan Chak

This project is a trial on encryption of images. We found that we can make good use of radon transform to convert the images to be 'unreadable’ data, and with random permutation key.


Although the images cannot be fully recovered due to inverse problem, the image is still visible for human after decryption.
